Why ZephyrOS?

Engineered to get out of your way and let you focus on what matters.

High Performance

Built on the strong foundation of Bazzite, experience reliable performance whether you're compiling code or gaming.

Privacy First

No telemetry, no tracking. Your data belongs to you. We believe in transparency and open-source integrity.

Fully Customizable

Make it yours. Built on the GNOME Desktop, experience a custom desktop entirely configurable from the extensions app. Everything you need to tinker comes pre-installed

Designed for Developers, Built for Everyone

Powerful tools for professionals, wrapped in an intuitive experience anyone can enjoy.

Pre-configured with essential dev tools for the modern developer (Git, VSCode, Brew).

Pre-installed popout terminal provided by ddterm.

Optimized for container-based workloads with Podman and DistroShelf pre-installed for easy setup.

Easily install software with Flatpak, Brew, ujust, and DistroShelf containers, or layer your own Fedora packages with rpm-ostree.

NVIDIA Drivers

Latest proprietary NVIDIA drivers pre-installed. No terminal commands, no blacklisting nouveau—just boot and play.
